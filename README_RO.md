# Brăila Turistică

Acest proiect reprezintă un site web static realizat în HTML, CSS și
JavaScript, având ca scop prezentarea principalelor obiective turistice
din municipiul Brăila.

## Conținutul proiectului

Site‑ul include:

-   Pagina principală (`index.html`)
-   Pagini dedicate pentru fiecare obiectiv turistic:
    -   Ceasul din Centru
    -   Turnul de Apă
    -   Moara Violatos
    -   Strada Regală
    -   Faleza Dunării
    -   Biserica Greacă (Sfinții Arhangheli)
-   Galerie foto
-   Pagină de contact
-   Fișiere de stil (`styles.css`, `page-styles.css`)
-   Script JavaScript (`script.js`)
-   Folder `images` cu imagini demonstrative

## Utilizare

1.  Descărcați proiectul pe calculator.
2.  Deschideți fișierul `index.html` într‑un browser web (Chrome, Edge,
    Firefox etc.).
3.  Navigați între pagini folosind meniul site‑ului.

## Publicare online (GitHub Pages)

Pentru publicarea site‑ului gratuit:

1.  Creați un repository public pe GitHub.
2.  Încărcați toate fișierele proiectului.
3.  Accesați **Settings → Pages**.
4.  Selectați:
    -   Source: Deploy from branch
    -   Branch: main
    -   Folder: /root
5.  Salvați modificările și așteptați generarea link‑ului.

## Observații

-   Imaginile pot fi înlocuite cu fotografii reale păstrând aceleași
